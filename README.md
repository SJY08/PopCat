# PopCat
HTML로 만든 간단한 팝캣

자습실에서 심심해서 갑자기 만든 무언가입니다
전에 친구한테서 js 코드를 뜯어 간단히 js를 아마추어식으로 배우고,
Css도 멘토링할때 만든 코드를 뜯어 js와 똑같이 공부했습니다.

처음 구상은 다음과 같습니다

1. 팝캣이 클릭되면 입을 뻐끔하게 하는 것
2. 팝캣 클릭 시 카운트업

사실 소리는 추가하기 귀찮아 안하려 했으나, 저의 html 기초를 가르쳐주신 분께서
소리를 넣어달라 하시기에 넣었습니다

하여 최종적으로는 다음과 같이 나오게 됐습니다.
1. 팝캣 클릭 시 이미지 변경
2. 팝캣 클릭시 카운트 업
3. 팝캣 클릭시 오디오 출력
